# IFA-Cartao - Confere Aí

Um site interativo e educativo que apresenta fatos curiosos sobre animais através de cartões flip que revelam informações ao passar o mouse.

## Como Usar

1. **Abra o site** - Execute o arquivo `index.html` em seu navegador
2. **Explore os cartões** - Passe o mouse sobre cada cartão para fazer o flip e descobrir novos fatos
3. **Frente do cartão** - Mostra uma imagem do animal e uma curiosidade inicial
4. **Verso do cartão** - Ao passar o mouse, revela outra informação interessante sobre o animal
5. **Aprenda** - Descubra fatos fascinantes de forma visual e interativa

## Características

- 🎴 **Cartões Interativos** - Flip com efeito 3D ao passar o mouse
- 🖼️ **Design Responsivo** - Interface limpa e intuitiva
- 📚 **Conteúdo Educativo** - Fatos curiosos e informativos sobre animais
- 🎨 **Tema Verde** - Paleta de cores naturais e agradáveis

## Tecnologias

- HTML5
- CSS3 (com animações 3D)
- Design responsivo
